# Portfolio Website

Static portfolio website deployed with GitHub Pages.

## Files

- `index.html`: page structure and content
- `styles.css`: layout and visual styles
- `script.js`: video playback interactions
- `image/`: website image assets
- `.nojekyll`: GitHub Pages configuration helper

## Deployment

The site can be served directly from the repository root with GitHub Pages:

`Settings` -> `Pages` -> `Deploy from a branch` -> `main` -> `/root`
